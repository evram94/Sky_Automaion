#####Requirements#####
1-Install Python 3
2-Add the chromedriver.exe folder path to your PATH in system environment variables
note : chrome driver attached is for Google Chrome version 80, if you have a higher or lower version please make sure to download its coresponding verison of the chromedriver 
3-Install Selenium (pip install selenium)
4-Please make sure that Sky.bat and Sky.py are in the same folder


#####How to run#####

Just double click Sky.bat

OR press win+R then type cmd and hit enter then cd /script DIR/ the run python Sky.py
